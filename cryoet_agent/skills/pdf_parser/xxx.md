# PDF Parser
This skill allows users to extract information from PDF files, particularly focusing on data processing details that may be relevant for cryo-EM analysis. The key usage of this skill is to extract data processing information from a methode paper, or a cryo-EM data processing workflow pdf file. The key information that we want to extract includes: software name, software version, parameters used for each step of the data processing workflow, and the order of the data processing workflow.
